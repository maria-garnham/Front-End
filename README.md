💻 Projeto Belle Pizzas
Este é um projeto desenvolvido para a avaliação de Front-End, na qual foi simulado um site para uma pizzaria.

🛠️ Tecnologias Utilizadas
HTML5 (Semântica e Acessibilidade)
CSS3 (Estilização e Feedback Visual)
JavaScript (Manipulação de DOM e Eventos)
📁 Estrutura do Projeto
O projeto foi organizado seguindo boas práticas de mercado: / 
├── index.html
└── src/ 
  ├── estilos/ (Estilos da página) 
  ├── scripts/ (Lógica de interação)
  └── imagens/ (Ativos visuais)

🚀 Funcionalidades Implementadas
Interatividade Visual: Cores e fontes do banner que são destacados quando passa o mouse.
Saudação Dinâmica: Mensagem de boas-vindas que muda conforme o horário local do usuário. Por exemplo, Se o usuário acessar antes das 12, a mensagem será: Bom dia! Pizza no café da manhã?. 
Cálculo em Tempo Real: Formulário que processa e exibe resultados assim que o usuário digita a quantidade de pessoas no rodizio.
Injeção de Conteúdo: Adição e exclusão de novos itens na tela utilizando manipulação de DOM.
